Parameters

sourceDir
The full path to the source directory.
wildcards
One or multiple wildcards or filenames to search for in sourceDir (default is *.aspx,*.ascx).
When using multiple wildcards, use comma as delimiter.
targetDir
The full path to the destination directory (default is same than sourceDir).
recurseSubDirs
True if subdirectories should be included (default is false).
overwriteExistingFiles
True if existing files should be overwriten (default is false).

Remarks

If the targetDir equals to sourceDir, the converted filenames get modified as follow: "(file).converted.(ext)"
For example: "default.aspx" (VB.NET) will be converted to "default.converted.aspx" (C#)
