Parameters

sourceDir
The full path to the source directory.
wildcards
One or multiple wildcards or filenames to search for in sourceDir (default is VSProjectConverter.DefaultWildcard).
When using multiple wildcards, use comma as delimiter. For example: a*.vbproj,b*.vbproj,c*.vbproj to convert all projectfiles beginning with a, b or c.
targetDir
The full path to the destination directory (default is same than sourceDir).
recurseSubDirs
True if subdirectories should be included (default is false).
overwriteExistingFiles
True if existing files should be overwriten (default is false).
fixNamespaces
True if enable the append/remove default-using-statements as defined in the FixNamespaces Property (default is false).
